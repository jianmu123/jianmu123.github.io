ES6中的箭头函数

  • 原来的函数写法：
    function sum(n,m){
    return n+m;
    }
  • 现在的ES6箭头函数写法：
    let sum=(n,m)=>{
    return n+m；
    }
    如果函数体中只有一行return，可以省略大括号。采用下面的写法
    let sum(n,m)=>n+m;
箭头函数举例：
function fn(n){
        return function(m){
                return n+m;
                }
        }
改为箭头函数：
        let fn=n=>m=>n+m;
